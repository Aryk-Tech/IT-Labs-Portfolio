## Networking Labs

Hands-on networking labs focused on practical IT skills including:

- IP addressing and subnet basics
- DHCP and DNS configuration concepts
- Network troubleshooting using command-line tools
- Packet analysis and connectivity testing

Tools used:
- Windows Command Prompt
- Wireshark
- Virtual Machines (VirtualBox)
